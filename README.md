# AudioPleer
Это бесплатный AudioPlayer в котором есть так как и удобство так и полезные функции которых нету в других плеерах, например режим рандомного прослушивания (все треки перемешивает, при повторном нажатии всё возвращает). Или 3 режима для особых коблочек в глазу (болезни) Tritanopia, Дейтеранопия и Протанопия! Music -> Playlist1... Туда музыку класть!
