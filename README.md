# 🎵 HalexMusic Player 
Бесплатный музыкальный плеер на Python с использованием библиотек tkinter и т.п.

## Возможности:
- Плейлисты
- Мини-плеер
- Режимы для дальтоников (Deuteranopia, Tritanopia, Protanopia)
- Shuffle
- Горячие клавиши
- Сохранение настроек
