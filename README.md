## 🎧 О ПЛЕЕРЕ

HalexMusic — это бесплатный аудиоплеер на Python с красивым интерфейсом, визуализатором и поддержкой плейлистов. Создан с нуля в рамках курса «Python с нуля до аудиоплеера».

---

## ✨ ВОЗМОЖНОСТИ

### 🎨 Интерфейс
- 8 цветовых тем (включая темы для дальтоников)
- Мини-плеер (всегда поверх окон, можно перетаскивать)
- Два режима визуализатора: прямоугольный (64 полоски) и круговой (как в YouTube Music)
- Поддержка длинных названий треков

### 🎵 Аудио
- Поддержка MP3 (через Pygame + Mutagen)
- Shuffle с сохранением оригинального порядка
- Режимы повтора: обычный / один трек / весь плейлист
- Горячие клавиши (Пробел, Стрелки, Esc, Ctrl+F, Ctrl+M)

### 📂 Плейлисты
- Автоматическое сканирование папок на рабочем столе
- Создание, удаление, переименование плейлистов
- Экспорт плейлистов в JSON
- Поиск треков по названию и исполнителю

### ✏️ Редактор тегов
- Изменение отображения названия и исполнителя (не трогает оригинальный MP3)
- Сохраняется между перезапусками

### 💿 Установка
- Установщик для Windows (Inno Setup)
- Портабл-версия (EXE)

## 🎵 КАК ДОБАВИТЬ МУЗЫКУ

1. Плеер автоматически создаст папку `Music` на рабочем столе
2. Внутри будут папки `Playlist1`, `Playlist2`, `Playlist3`
3. Скопируй свои MP3-файлы в любую папку
4. Нажми кнопку «🔄» в плеере для обновления

| Клавиша | Действие |
|---------|----------|

| → | Следующий трек |
| Esc | Мини-плеер |
| Ctrl+F | Поиск |
| Ctrl+M | Переключить мини-плеер |
